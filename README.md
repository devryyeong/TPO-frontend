## TPO / 시간,장소,상황 기반 상품 큐레이션 서비스
###### 기간 : 5월 17일 ~ 6월 26일 
- React + Typescript + PWA + Styled-Components
- 시간(Time), 장소(Place), 상황(Occasion)에 맞게 나에게 맞는 옷차림 추천

### 기능 
- 시간(Time), 장소(Place), 상황(Occasion), 입력 및 회원가입 페이지
- 상품 추천 및 큐레이션 페이지
- 날씨 OpenAPI 활용
